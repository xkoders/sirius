# Actions

Action components provide interactive elements that allow users to perform actions within your application.

## Button {{ anchor: false }}

Buttons are essential interactive elements that allow users to perform actions. Our button component supports multiple variants, sizes, and states.

function MyComponent() {

### Button Props

<Properties>
  <Property name="variant" type="'primary' | 'secondary' | 'outline' | 'none'">
    The visual style variant of the button
  </Property>
  <Property name="size" type="'small' | 'medium' | 'large' | 'slim' | 'none'">
    The size of the button
  </Property>
  <Property name="disabled" type="boolean">
    Whether the button is disabled
  </Property>
  <Property name="loading" type="boolean">
    Whether the button is in loading state
  </Property>
  <Property name="fullwidth" type="boolean">
    Whether the button should take full width
  </Property>
  <Property name="outline" type="boolean">
    Whether to use outline style
  </Property>
  <Property name="onClick" type="() => void">
    Click handler function
  </Property>
</Properties>
