# Forms

Form components provide interactive elements for user input and data collection.

## TextField {{ anchor: false }}

Text fields allow users to input text data with various configurations and validation states.

<EnhancedCodeGroup 
  component="TextField" 
  componentProps={{ label: 'Email', placeholder: 'Enter your email' }}
  showPreview={true}
  frameworks={['react', 'vue']}
>

```tsx {{ title: 'React' }}
import { TextField } from '@adersolutions/sirius-react'

function MyComponent() {
  const [value, setValue] = useState('')
  
  return (
    <div className="space-y-4">
      <TextField 
        label="Email" 
        placeholder="Enter your email"
        value={value}
        onChange={setValue}
      />
      <TextField 
        label="Password" 
        type="password"
        error="Password is required"
      />
      <TextField 
        label="Multiline" 
        multiline={3}
        placeholder="Enter description"
      />
    </div>
  )
}
```

```vue {{ title: 'Vue' }}
<template>
  <div class="space-y-4">
    <TextField 
      label="Email" 
      placeholder="Enter your email"
      v-model="email"
    />
    <TextField 
      label="Password" 
      type="password"
      error="Password is required"
    />
    <TextField 
      label="Multiline" 
      :multiline="3"
      placeholder="Enter description"
    />
  </div>
</template>

<script setup>
import { ref } from 'vue'
import { TextField } from '@adersolutions/sirius-vue'

const email = ref('')
</script>
```

</EnhancedCodeGroup>

## Checkbox {{ anchor: false }}

Checkboxes allow users to select one or more options from a set.

<EnhancedCodeGroup 
  component="Checkbox" 
  componentProps={{ label: 'Accept terms', checked: true }}
  showPreview={true}
  frameworks={['react', 'vue']}
>

```tsx {{ title: 'React' }}
import { Checkbox } from '@adersolutions/sirius-react'

function MyComponent() {
  const [checked, setChecked] = useState(true)
  const [indeterminate, setIndeterminate] = useState(false)
  
  return (
    <div className="space-y-4">
      <Checkbox 
        label="Accept terms and conditions"
        checked={checked}
        onChange={setChecked}
      />
      <Checkbox 
        label="Subscribe to newsletter"
        checked={false}
      />
      <Checkbox 
        label="Indeterminate state"
        indeterminate={indeterminate}
        onChange={() => setIndeterminate(!indeterminate)}
      />
      <Checkbox 
        label="Disabled checkbox"
        disabled
        checked={true}
      />
    </div>
  )
}
```

```vue {{ title: 'Vue' }}
<template>
  <div class="space-y-4">
    <Checkbox 
      label="Accept terms and conditions"
      v-model="checked"
    />
    <Checkbox 
      label="Subscribe to newsletter"
      :checked="false"
    />
    <Checkbox 
      label="Indeterminate state"
      :indeterminate="indeterminate"
      @change="indeterminate = !indeterminate"
    />
    <Checkbox 
      label="Disabled checkbox"
      disabled
      :checked="true"
    />
  </div>
</template>

<script setup>
import { ref } from 'vue'
import { Checkbox } from '@adersolutions/sirius-vue'

const checked = ref(true)
const indeterminate = ref(false)
</script>
```

</EnhancedCodeGroup>
