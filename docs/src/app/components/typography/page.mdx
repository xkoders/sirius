# Typography

Typography components provide semantic HTML elements with consistent styling and spacing.

<Note>
Typography components automatically handle responsive design and accessibility requirements. They provide semantic meaning and consistent visual hierarchy.
</Note>

## Heading {{ anchor: false }}

Headings provide semantic structure for page and section titles.

## Typography Best Practices

### Semantic Structure
- Use `Heading` components for document structure (h1-h6)
- Use `Title` for page-level headings
- Use `Subtitle` for section descriptions
- Use `Text` for body content

### Accessibility
- Headings provide semantic structure for screen readers
- Proper heading hierarchy improves navigation
- Consistent spacing improves readability

### Responsive Design
- Typography components automatically adapt to screen sizes
- Consistent spacing across all breakpoints
- Optimized line heights for readability
