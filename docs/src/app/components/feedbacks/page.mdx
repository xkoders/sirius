# Feedbacks

Feedback components provide visual indicators and notifications to inform users about the current state of the application.

## Badge {{ anchor: false }}

Badges are small status indicators that provide quick visual information about an item's state.

<EnhancedCodeGroup 
  component="Badge" 
  componentProps={{ children: 'New', status: 'success' }}
  showPreview={true}
  frameworks={['react', 'vue']}
>

```tsx {{ title: 'React' }}
import { Badge } from '@adersolutions/sirius-react'

function MyComponent() {
  return (
    <div className="flex gap-2 flex-wrap">
      <Badge status="default">Default</Badge>
      <Badge status="info">Info</Badge>
      <Badge status="warning">Warning</Badge>
      <Badge status="critical">Critical</Badge>
      <Badge status="success">Success</Badge>
      <Badge status="pending">Pending</Badge>
    </div>
  )
}
```

```vue {{ title: 'Vue' }}
<template>
  <div class="flex gap-2 flex-wrap">
    <Badge status="default">Default</Badge>
    <Badge status="info">Info</Badge>
    <Badge status="warning">Warning</Badge>
    <Badge status="critical">Critical</Badge>
    <Badge status="success">Success</Badge>
    <Badge status="pending">Pending</Badge>
  </div>
</template>

<script setup>
import { Badge } from '@adersolutions/sirius-vue'
</script>
```

</EnhancedCodeGroup>

### Badge Props

<Properties>
  <Property name="status" type="'default' | 'info' | 'warning' | 'critical' | 'success' | 'pending'">
    The status variant of the badge
  </Property>
  <Property name="type" type="'incomplete' | 'halfComplete' | 'complete' | 'dashed'">
    The type of badge (for progress indicators)
  </Property>
  <Property name="icon" type="IconType">
    Optional icon to display in the badge
  </Property>
</Properties>

## Toast {{ anchor: false }}

Toast notifications provide temporary feedback messages to users.

<EnhancedCodeGroup 
  component="Toast" 
  componentProps={{ content: 'Operation completed successfully', type: 'success' }}
  showPreview={true}
  frameworks={['react', 'vue']}
>

```tsx {{ title: 'React' }}
import { Toast } from '@adersolutions/sirius-react'

function MyComponent() {
  return (
    <div className="space-y-2">
      <Toast content="Success message" type="success" />
      <Toast content="Error message" type="error" />
      <Toast content="Info message" type="info" />
      <Toast content="Warning message" type="warning" />
    </div>
  )
}
```

```vue {{ title: 'Vue' }}
<template>
  <div class="space-y-2">
    <Toast content="Success message" type="success" />
    <Toast content="Error message" type="error" />
    <Toast content="Info message" type="info" />
    <Toast content="Warning message" type="warning" />
  </div>
</template>

<script setup>
import { Toast } from '@adersolutions/sirius-vue'
</script>
```

</EnhancedCodeGroup>

### Toast Props

<Properties>
  <Property name="content" type="string | ReactNode">
    The content to display in the toast
  </Property>
  <Property name="type" type="'success' | 'error' | 'info' | 'warning' | 'default'">
    The type of toast notification
  </Property>
  <Property name="duration" type="number">
    Duration in milliseconds before auto-dismiss (default: 4500)
  </Property>
  <Property name="onDismiss" type="() => void">
    Callback when toast is dismissed
  </Property>
</Properties>
